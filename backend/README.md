# Backend API

## Setup Instructions

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the server:
   ```bash
   node server.js
   ```

3. API Endpoints:
   - **POST /api/admin/register** → Register an admin.
   - **GET /api/institutes** → Fetch unique institute names.
   - **POST /api/admin/login** → Login as an admin.
